    • 如何高效的除甲醛
    2018-07-31 / 1 min read

    需要声明的是，此方法我还未完全实验成功，但我觉得已经初有成效

    旧方法

    1. 活性炭
    2. 绿植
    3. 开窗通风
    4. 喷洒二氧化钛
    5. 活性炭晶石

    从别人的方法总结而来

    1. 甲醛是一种无色有刺激性气味的存在，至于危害，百度也好，Google也好，均能找到资料
    2. 甲醛在高温（30度以上），高湿（70+）时释放最快，高温低湿度，甲醛释放不出来，低温高湿，则溶于水。所以高温高湿是非常必要的，所以夏天是最好的除甲醛的季节。
    3. 准备工业型电风扇一台（带过温保护），离心风机和配套的管子，加湿器（如果当地的湿度不够），空调（如果当地温度不够）
    4. 将房间里的温度和湿度弄到之前第二条列出的条件，打开电扇和离心风机，一边抽风，一边吹风，尽量选择在白天进行，因为白天温度高，更容易将甲醛吹出去或者吸出去
